📡 Quick API Reference

Try these in a terminal (curl) or Postman:

# Ingest a single call (priority 1 = VIP)
curl -X POST http://localhost:8080/api/v1/calls \
     -H "Content-Type: application/json" \
     -d '{"callerId":"+1-555-0100","priority":1}'

# Bulk ingest 3 calls
curl -X POST http://localhost:8080/api/v1/calls/bulk \
     -H "Content-Type: application/json" \
     -d '[{"callerId":"+1-555-0101","priority":2},{"callerId":"+1-555-0102","priority":5},{"callerId":"+1-555-0103","priority":1}]'

# Queue stats
curl http://localhost:8080/api/v1/calls/stats

# Register an agent
curl -X POST http://localhost:8080/api/v1/agents \
     -H "Content-Type: application/json" \
     -d '{"name":"Agent Charlie"}'

# List agents
curl http://localhost:8080/api/v1/agents

# ACD metrics snapshot
curl http://localhost:8080/api/v1/metrics/snapshot

# Spring Actuator health
curl http://localhost:8080/actuator/health
